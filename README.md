Compose weather forecast app which uses VisualCrossing API to load weather data and OpenWeatherMap API for geocoding. Users can search the location they want or get forecast info of their current location if location permission is granted.

Resources used: Jetpack Compose for building the UI, Coroutines, Room, Retrofit
